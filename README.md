# Calendar System User’s Manual
## Introduction
```
本產品為曆表系統，功能有顯示該月月曆、輸入西元紀年轉
換干支紀年、生肖、計算天數、計算日期四種功能。
```
## Run
```
Eclipse 4.15
```
## Error Example
```
1. 請輸入有效指令或關閉系統
2. For function ChineseYear : Invalid input format For function parsedate : Invalid index
```
